# Supporting Evox Development

Thank you for your interest in supporting the Evox framework! Your contributions help ensure the project's continued development, maintenance, and improvement.

## Why Donate?

Evox is an open-source project developed and maintained by volunteers. While the framework is free to use for everyone under the permissive Apache License 2.0, ongoing development requires time, effort, and sometimes financial resources for infrastructure, tools, and community support.

Your support helps us:
- Maintain and improve the core framework
- Develop new features and enhancements
- Fix bugs and security issues promptly
- Create documentation and learning resources
- Build a vibrant community around the project

## Donation Options

### 1. Direct Contributions to Evox

While we're setting up formal donation channels, you can:

- **Contribute code, documentation, or translations** - often the most valuable form of support!
- **Report bugs and suggest features** to help improve the framework
- **Spread the word** about Evox in your community

### 2. Future Donation Platforms

We're working on establishing:
- GitHub Sponsors organization for Evox
- OpenCollective page for transparent funding

Stay tuned for updates on these channels!

### 3. Support Related Projects

Since Evox is built on top of excellent open-source foundations, you can also support:

#### Core Dependencies

Evox heavily relies on these essential projects:

- **FastAPI** - High performance, easy to learn, fast to code, ready for production API framework
- **Typer** - Library for building CLI applications
- **Pydantic** - Data validation and settings management using Python type annotations
- **Uvicorn** - Lightning-fast ASGI server implementation
- **HTTPX** - Fully featured HTTP client for Python 3

#### Additional Dependencies

Other important projects that make Evox possible:

- **Jinja2** - Modern and designer-friendly templating language for Python
- **Tomli/TOML** - Library for parsing TOML configuration files
- **Aiosqlite** - Asynchronous interface to SQLite
- **Webbrowser** - Standard library module for web browser integration

Supporting these projects helps ensure the entire ecosystem remains healthy and continues to evolve.

## Corporate Sponsorship

For companies using Evox commercially, we encourage you to consider supporting the project's sustainability. While our Apache License 2.0 imposes no obligations, we kindly suggest that organizations deriving significant value from Evox consider contributing back to its development.

Corporate sponsors receive:
- Recognition in our documentation and README
- Priority support for critical issues
- Early access to new features (when applicable)
- Influence in roadmap discussions

## How Your Support Is Used

Donations and sponsorships are typically used for:
1. **Development time** - Paying maintainers and contributors for their work
2. **Infrastructure** - Hosting, CI/CD, testing environments
3. **Community** - Events, meetups, educational content
4. **Tools and services** - Software licenses, productivity tools
5. **Documentation** - Professional writers, translators, designers

## Questions?

If you have questions about supporting Evox or would like to discuss corporate sponsorship, please reach out through our GitHub discussions or contact the maintainers directly.
